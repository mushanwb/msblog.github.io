windows & linux 上安装 java

Posted on 2020-03-28 | In java

windows 上安装 java 开发环境

**下载 JDK

Java Development Kit（JDK）是太阳微系统针对 Java 开发人员发布的免费软件开发工具包（SDK，Software development kit）。

百度或goole搜索 oracle jdk，一般在搜索第一个出现会 Java SE 下载 网站，找到 Java SE 8u开头的版本，这个版本是用的最多的版本，相对成熟稳定。进版本后根据电脑的配置选择安装包，我的电脑是windows 64位的，所以我选择下载 jdk-8u241-windows-x64.exe

点击链接后可能会出现oracle需要登录，如果不想注册oracle账号的话，可以去百度或者google搜索 oracle jdk 账号，一般会有人给出的，当然直接注册oracle 账号也是很简单的。登录完后就直接下载了。

安装 JDK

下载完 JDK 后，安装的步骤很简单，直接双击一直点下一步就可以了，其中建议不要更改文件目录，如果对文件操作很熟的话可以自由发挥。

安装完后就开始设置环境变量，虽然开发一般都是IDEA中完成，但是避免有些时候需要用到命令行，所以还是设置一下环境变量。

在windows上设置环境变量：

1、右键点击我的电脑，单击属性，单击高级系统设置，选择高级，然后单击环境变量

选择环境变量后，在环境变量中选择Path，然后单击编辑，将 JDK 的安装目录添加进去，如果在安装 JDK 的时候没有修改过文件目录，那默认的安装目录为 C:\Program Files\Java\jdk1.8.0_241

注意：在编辑Path中，会有一个系统给的 JDK 环境变量（如果是刚刚安装的 JDK，一般会在第一个位置），将我们自己设置的环境变量地址放到这个系统给的地址前面（必须做的），这样就会覆盖系统默认给的环境变量。还有一点就是要在默认的安装地址后面加一个 bin 目录，即：C:\Program Files\Java\jdk1.8.0_241\bin，将这个地址添加进去，一般应用程序在这个bin目录里。添加完需要重启电脑。

重启电脑后，在windows的终端（window + R，输入 cmd 确定就可以打开了）中，输入命令

java -version 回车，出现以下内容，则代表环境安装成功了。

下载 IDEA

去 地球最强编辑器的官网 上下载java的IDEA，也就是开发所用到的编辑器。找到图标为 I J 样式的点进去，点进去后不管是windows还是mac都会有两种版本可供选择，一种是 Ultimate 旗舰版和 Community 社区版，一般对初学者来说使用 Community 社区版已经足够用了，Ultimate 旗舰版是需要收费的，费用还比较高，一年大约一千人民币。

直接点击下载就可以了，如果下载的右边有选项的话，就选择 windows,JBR 8 (.EXE)。下载完后直接点下一步就可以了，如果不知道选项是什么，就点下一步用默认的就行了。

安装完后在windows的搜索里直接搜索 Intel就会出来，双击后会出现一个提示框，问需不需要导入，默认是不需要，直接点击 ok ，然后勾选协议，点击 Continue ，然后会出现一个问你要不要发送邮箱帮他们做改进的一个信息，点击 Don't send ，然后选择主题，有黑边两种主题，一般默认是黑色，看自己喜好，点击 Next: ，然后就是可以选择自己喜欢的插件，对于初学者来说，暂时可以不选，点击 Next: ，在继续点击 Next： ，到这里IDEA就算是完成了。

在 Linux 上安装 Java 环境

下载 JDK

一般我使用的 CentOs，ubuntu 或者其他的可自行google。

依旧是去 JDK 的官网，按照下载 windows 版本的方式，这次选择 Linux 64 的版本，记住一点要选择 .tar.gz 压缩包版的，点击下载，还是需要登录 oracle 账号，登录完后就开始下载，进到下载的页面，复制下载页面上的那个链接也就是下载链接(右键点击链接，选择复制链接地址)，然后在 Linux 终端里敲命令下载：

wget https://download.oracle.com/otn/java/jdk/8u241-b07/1f5b5a70bf22433b84d0e960903adac8/jdk-8u241-linux-x64.tar.gz?AuthParam=1585453028_4c84b37ce15ddcebdddb52314dff266d

（这是我的下载链接地址）

注意：这个下载链接要马上使用，有实效的，实效还比较短，每次下载链接都可能不一样。

下载完后就会出现一个安装包，如果觉得名字不太对或者太长了，就重命名一下：

mv jdk-8u241-linux-x64.tar.gz\?AuthParam\=1585453028_4c84b37ce15ddcebdddb52314dff266d jdk-8u241-linux-x64.tar.gz

我这里已经将安装包重命名为 jdk-8u241-linux-x64.tar.gz

开始解压缩：

tar -zxvf jdk-8u241-linux-x64.tar.gz

解压完后会出现一个 jdk1.8.0_241开头的文件夹，这时候 纯绿色jdk 已经安装成功了。看一下jdk版本信息：

jdk1.8.0_241/bin/java -version

出现一下类似信息则ok了：

绿色版jdk：只需要把这个解压好的文件夹放在任何一个地方，然后使用它的全路径或者JAVA_HOME环境变量就能用了。

因为这是纯绿色安装的，如果想卸载的话，直接删除文件夹就可以了。

概念补充：JRE JDK JAVA_HOME 术语解释

JRE 的全程为 Java Runtime Environment，是 Java的运行环境`，可以运行 Java编译后的程序。

JDK 的全程为 Java Development Kit，是 Java的开发和运行环境，可以简单初略的理解为 JDK = JRE + Javac。

Javac是 Java语言的 编译器，Java代码写完后需要编译后运行，但是编译过程需要在 JDK 上完成，编译完后可以在 JRE 上运行。

JAVA_HOME 简单理解就是 JDK安装的文件夹，可以将安装全路径设置为 JAVA_HOME 环境变量
