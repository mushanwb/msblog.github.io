laravel redis队列推送

Posted on 2019-11-26 | In php

环境要求：

  • redis
  • laravel 5.5+

redis配置

项目中使用 Composer 安装依赖

在laravel得配置文件.env中配置redis

QUEUE_DRIVER=redis

REDIS_HOST=127.0.0.1

REDIS_PASSWORD=null

REDIS_PORT=6379

参数解释：

QUEUE_DRIVER：队列驱动选择，参数值为database（数据库）或者redis，后面会小提一下

REDIS_HOST：redis的ip，一般都在本地

REDIS_PASSWORD：redis的密码，没有设置密码就为默认的null

REDIS_PORT：redis的端口号，如果没有改端口号，则默认为6379
